# Planned Features v4 (Challenge & Polish)

## Goal
Focus on game balance, increasing difficulty, and adding "Rush Hour" mechanics to prevent the game from feeling too "cozy" or static.

## 1. Gameplay Challenge Features (Fix_First #5)
- [ ] **Rush Hour Event**:
    - Between 8:00 AM and 10:00 AM, customer spawn rate doubles (or patience decays faster).
    - Visual cue: "Rush Hour!" notification.
- [ ] **Ingredient Spoilage**:
    - Milk and Beans degrade if held too long (or overnight?).
    - Cost of waste management.
- [ ] **Machine Breakdowns**:
    - Random chance for Espresso Machine or Grinder to jam.
    - Mini-game to "Repair" (Click/Hold mechanics from Cart Designer?).
- [ ] **Complex Orders**:
    - Unlocks at higher reputation.
    - "Half-caf, soy milk, extra foam, 2 pumps vanilla".
    - Requires precise recipe steps.

## 2. Mechanic Expansions
- [ ] **Weld More** (Fix_First #10):
    - Add welding mechanic to repairs or decorations.
- [ ] **Recipe Book**:
    - Visible list of unlocked drinks.
    - Locked slots for complex drinks.

## 3. UI/UX Improvements
- [ ] **Supply Alerts**: Popup when supplies hit 0.
- [ ] **Wallet in Shop**: Display current cash in Shop modal.
- [ ] **Customer Sentiment**: Mood swings logic.
- [ ] **Order Popup**: Visual order bubble.
